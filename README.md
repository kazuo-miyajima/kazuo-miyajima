> **抜粋（ニューラルネットワークの解説）:**
> 「入力層からデータが流れる際、『重みづけ』という操作がなされます。重みは、データの関連の強さや重要度を表すパラメーターで、この値が大きいほど、そのデータの影響力が増幅されるということです。（中略）この多層化こそが、現在のディープラーニングの飛躍的な進化を支えています。」
